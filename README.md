# industry project server
This is the back-end server side part of 24-hour hackathon ft. Microsoft (industry partner) on Feb 13-14, 2025.

Team: Aaron Mahoney, Helana Youssef, Julia Babicheva, Kateryna Zabrodska, Nadia Kasim

## DB Setup Instructions

0. Make sure all the dependencies are installed.
1. Replace the placeholders in `.env.sample` file and turn it into an `.env`.
2. Create a DB called `XYZ`.
3. Run `npm run db:migrate` in terminal to create the tables.
4. Run `npm run db:seed` in terminal to populate the tables.
